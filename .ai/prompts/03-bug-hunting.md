# 🐞 Bug Hunting Prompt

**Copy and paste the text below into your AI chat when encountering a stubborn bug:**

```text
We have a stubborn bug. Please enter "Reflexion Mode".
1. STOP writing code. Do not attempt another fix yet.
2. Analyze the root cause. What assumption did we make that was wrong?
3. Review related API Contracts or documentation.
4. Propose a single, targeted fix.
5. Once fixed, log this lesson in `.ai/REFLECTIONS.md`.
```
